								< HOW TO USE >


*[Administrator]
----------------------------------------------------------------------------------
1.Get the Folder and the full project directory in Eclipse
2.Find the launcher folder inside src folder
3.Run the Main.java in the launcher folder

Use your login information to access the functions:

1.Add users(including new administrators as well) or identities
2.Update user/identity
3.Delete user/identity
4.Search user/identity


*[Visitor]
----------------------------------------------------------------------------------
1.Open the html page (src/index.html)
2.Scroll and type in the name of identity to search

This will allow Visitor relevant function ONLY.
